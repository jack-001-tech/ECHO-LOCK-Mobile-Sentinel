# ECHO-LOCK Mobile Sentinel (360°)

## AI-Powered Autonomous Mobile Surveillance Platform

ECHO-LOCK Mobile Sentinel (360°) is an intelligent surveillance system designed to proactively detect, investigate, track, and document security incidents using artificial intelligence and computer vision.

Unlike traditional CCTV systems that passively record events, ECHO-LOCK is designed to actively respond to suspicious activity by combining autonomous movement, object detection, target tracking, and automated evidence collection into a unified platform.

---

# Project Objectives

- Detect suspicious activity in real time.
- Investigate detected events autonomously.
- Track multiple objects using AI vision.
- Generate digital evidence automatically.
- Improve surveillance efficiency through intelligent automation.
